Unsupported Media Type 415 错误码解析

  • 错误类型1：

"status":415 && "error":"Unsupported Media Type"

原因：没有使用JSON.stringify(rowData)方法将请求参数转化为json字符串

  • 错误类型2：

"status":400 && "error":"Bad Request"

原因：未使用[contentType : ‘application/json’,]配置指定请求参数格式。
